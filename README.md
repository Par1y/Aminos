# Aminos

The half-auto browser with AI assistants.

***based on [ChromeDriver](https://developer.chrome.com/docs/chromedriver/) & [Dify](https://dify.ai)***

## How to use

1. Open dify and install `dify_plugin/chromedriver.difypkg` or [load the plugin in debug mode](https://docs.dify.ai/plugin-dev-zh/0222-tool-plugin).

2. Launch a ChromeDiver Service via `launch.py` or do it yourself.

3. Configure plugin settings.

4. Load the `Aminos.yml` as DSL.

5. Adjust anything you want, and try it out.
